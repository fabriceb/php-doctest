		getExamples (line 193)
	
 
	
	

Returns an array of Example objects after parsing the string.


Line numbers are zero-based, because it's most common in doctests  that nothing interesting appears on the same line as the opening  comment string, and so the first interesting line is called "line 1"  then. The optional name argument is only used for error messages.
